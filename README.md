# 💳 Credit Card Fraud Detection

A Machine Learning web application that predicts whether a credit card transaction is **Fraudulent** or **Normal** using a trained classification model and a Flask web interface.

## 🌐 Live Demo

https://credit-card-fraud-detection-xyfg.onrender.com

## 🚀 Features

* Real-time fraud detection
* Machine learning model using Scikit-learn
* Web interface built with Flask
* Deployed globally using Render

## 🛠 Tech Stack

Python • Flask • Scikit-learn • NumPy • HTML • CSS • GitHub • Render

## 📂 Project Structure

```
credit-card-fraud-app
│
├── app.py

```

```
git clone https://github.com/aswini-barad/Credit-card-fraud-detection.git
cd Credit-card-fraud-detection/credit-card-fraud-app
pip install -r requirements.txt
```
